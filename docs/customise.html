How to install Close

Installing your icons with Mod Manager

  • Open Roblox Studio Mod Manager (or download it here)
  • Click 'Edit Class Icons' to open the class icon editor
  • Click 'Open Class Icon Folder'
  • Drop in all the icons from the 'ModManagerIcons' folder that came with your download.

Installing your icon set manually

  • Find your Roblox folder (e.g. on Windows, this is %AppData%/Local/Roblox/)
  • Go into /Versions/, then go into the latest version folder (check date modified!)
  • Navigate to the /content/textures/ folder
  • Drop in the 'ClassImages.png' image that came with your download.

Something to note: if you're installing manually, you'll have to redo this after every Roblox update.
